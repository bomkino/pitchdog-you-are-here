# Learnings

- The useful question is often “what moves this forward?” rather than “what deck should I make?”
- One primary move is more usable than a long diagnostic report.
- Structured route facts support deterministic guidance; prose the engine cannot read only performs intelligence.
- Expert depth works when the user consents and can see the reward.
- Access, rights, readiness, and evidence can outrank document production.
- Conditional facts must disappear when the situation that made them relevant changes.
- A compact on-screen answer can still export a complete, inspectable record.
- Quiet guidance above a choice set lowers uncertainty without turning the journey into instructions about instructions.
